# Pysplore-Mini-OS
Simple creativity, mini-game, utility app suite. Fully local Python OS app, no dependencies, besides GUI. Standard Python library only. Offline, royalty-free, open source. 

Includes: Music maker (Mini DAW), Media Player, Ambience Generator, Paint Tool,
Journal / Notepad, A clock with a stopwatch, calendar, timer, and alarm
calculator, chess, checkers, and solitaire.
